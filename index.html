Générateur de mots de passe sécurisés

Un clic, un mot de passe !

Sélectionnez les critères de génération de votre mot de passe sécurisé :

Sécurisé ?

Oui, le mot de passe vous est parvenu à l'aide de JavaScript, et n'est donc pas présent dans le code source de la page. De plus, il est généré de façon totalement aléatoire en fonction des critères que vous avez choisis.

Sachez également qu'aucune donnée personnelle ni trace des mots de passe créés ne sont stockés sur ce site.

Un mot de passe sécurisé...

  • contient au minimum 10 caractères ;
  • contient des majuscules ;
  • contient des minuscules ;
  • contient des chiffres ;
  • contient des caractères spéciaux comme * $ % ! : ; . , ? & ( - _ ) = # { [ | @ ] } ;
  • ne ressemble pas à l'un de vos mots de passe ;
  • n'est pas partagé entre plusieurs comptes ;
  • n'est pas divulgué à qui que ce soit ;
  • ne contient pas votre nom ;
  • ne contient pas votre prénom ;
  • ne contient pas votre identifiant ;
  • ne contient pas le nom d'un de vos proches ;
  • ne contient pas votre date de naissance ou celle d'un proche ;
  • ne contient pas de mots communs ;
  • ne contient pas une suite de caractères venant de votre clavier comme azerty, qwertyuiop ou 123456789.
